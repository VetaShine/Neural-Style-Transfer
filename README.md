# Style-Transfer-Telegram-Bot
Проект по предмету "Объектно-ориентированное программирование" - телеграм бот стилизации изображений. Стилизация изображений происходит за счет предварительно подготовленной свёрточной нейронной сети выделения признаков изображений - VGG16 , обученной мною для пяти стилей. Обученные веса модели можно скачать по ссылке: https://disk.yandex.ru/d/0HQSxoOTknugWw
